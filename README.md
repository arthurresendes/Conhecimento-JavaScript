## 📌 Descrição

Repositório dedicado ao aprendizado abrangente de JavaScript moderno, com enfoque em conceitos fundamentais, manipulação do DOM e integração com HTML e CSS. Documentação prática de exercícios, projetos pequenos e padrões de desenvolvimento para aplicações web.

Este projeto oferece uma base sólida para quem deseja dominar JavaScript e construir aplicações web dinâmicas e interativas.

## ✨ Funcionalidades

- 📚 **Fundamentos de JavaScript**: Variáveis, funções, escopos e closures
- 🎨 **Manipulação do DOM**: Seleção, criação e modificação de elementos HTML
- ⚡ **Eventos e Interatividade**: Event listeners, delegação de eventos
- 🎯 **ES6+ Features**: Arrow functions, template literals, destructuring, classes
- 📦 **Gerenciamento de Pacotes**: Configuração com npm e package.json
- 🖥️ **Integração HTML/CSS**: Projetos web completos com estilo
- 🧪 **Exercícios Práticos**: Desafios e soluções aplicadas

## 🛠️ Tecnologias

- **JavaScript** (ES6+)
- **HTML5**: Estrutura semântica
- **CSS3**: Estilos avançados e layouts responsivos
- **Node.js**: Runtime para execução
- **npm**: Gerenciador de pacotes
