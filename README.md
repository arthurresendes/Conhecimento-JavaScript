# Aprendizado JavaScript

Este repositório é focado no estudo, prática e aperfeiçoamento em **JavaScript**, contendo exemplos, projetos pequenos e experimentações com HTML e JavaScript puro.

---

## Estrutura do Projeto

Você encontrará exemplos práticos para:

- Manipulação do DOM
- Eventos de usuário
- Sintaxe moderna do JavaScript (ES6+)
- Exercícios de lógica e algoritmos
- Integração JavaScript + HTML para front-end básico
- Pequenos projetos como calculadoras, animações e interações

Os arquivos HTML servem de apoio para o código JS, permitindo testar na própria máquina e visualizar a execução.

---

**Este repositório é ideal para quem deseja aprender JavaScript do zero com exemplos simples e práticos, explorando desde lógica básica até interatividade no navegador!**
